這是一個可自訂的棒球計分版專案。

您可以透過這個工具，即時建立、管理並顯示一場棒球比賽的詳細狀態。它非常適合用於模擬比賽、記錄小型賽事，或作為棒球相關應用的前端介面。

## 主要功能

* **隊伍得分：** 自由設定主隊與客隊的目前得分。
* **比賽局數：** 輕鬆切換目前的比賽局數（例如：五局上、九局下）。
* **投打情況：** 即時更新好球數 (S)、壞球數 (B) 和出局數 (O)。
* **壘包狀態：** 清楚顯示一、二、三壘上是否有跑者。
* **完全自訂：** 所有比賽狀態皆可手動即時調整。
